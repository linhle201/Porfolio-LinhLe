About Me!

Hi! I'm Linh Le, a passionate web developer and designer dedicated to bringing ideas to life with intuitive, user-friendly web experiences. Currently, I’m a second-year student at Nova Scotia Community College, where I’ve had the opportunity to dive deep into both front-end and back-end development through a variety of hands-on projects.

I specialize in modern web technologies, including HTML, CSS, JavaScript, and React, and I’m proficient in design tools such as Figma, Bootstrap, and Tailwind. My goal is to build responsive, visually captivating websites and applications that not only look great but also offer seamless functionality, ensuring users can easily achieve their goals.

I’m always exploring new technologies and excited to collaborate on projects that challenge creativity and innovation. In addition to web development, I have experience with JavaScript, Java, and app development frameworks like Laravel, which broadens my ability to tackle diverse technical challenges.

Driven by a strong work ethic, I’m passionate about contributing to environments that foster collaboration, continuous learning, and the exchange of ideas. Whether working independently or as part of a team, I’m eager to create solutions that leave a lasting impact.

Project 1

Real-Time Customer Feedback App for Restaurants

I created an app to gather customer feedback in real-time using QR codes and forms, allowing restaurant managers to instantly assess performance.

Technologies Used:
  • Frontend: HTML, CSS, Bootstrap for responsive design
  • Backend: PHP for server-side logic
  • Containerization: Docker for easy deployment and management
  • Database: SQL for storing and querying feedback data

This app enables instant analysis of customer feedback, helping restaurants improve service and customer experience efficiently.

Project 3

Three-in-a-Row Puzzle (JavaScript)

I developed a 3-in-a-Row puzzle game using pure JavaScript.

The puzzle grid is dynamically generated from remote JSON data and displayed in an HTML table.

Players can cycle through cell states, check puzzle progress, and highlight incorrect cells.

The app supports puzzles of varying sizes, and the solution is built with unobtrusive JavaScript for clean, maintainable code.

Project 4

Transit Tracker

In this project, I created a real-time Transit Tracker using the Leaflet.js library to display vehicle positions on a map.

  • Real-time Map:

    Displays vehicle positions on an interactive map.

  • Auto-Refresh:

    Updates data every 15 seconds for live tracking.

  • Custom Icons & Rotation:

    Shows custom vehicle icons and rotates them based on travel direction.

  • Popups:

    Displays additional vehicle information on click.
